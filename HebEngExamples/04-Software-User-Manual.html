Chapter 4: Account Security Settings
English

This chapter explains how to configure the security features available in SecureConnect Pro, including multi-factor authentication (MFA), session management, and access permissions.

עברית

פרק זה מסביר כיצד להגדיר את אפשרויות האבטחה הזמינות ב-SecureConnect Pro, כולל אימות רב-גורמי (MFA), ניהול הפעלות והרשאות גישה.

4.1 Enabling Multi-Factor Authentication
English

To configure MFA, navigate to Account Settings > Security and click Enable MFA. You will be prompted to scan a QR code using an authenticator app of your choice, such as Google Authenticator, Microsoft Authenticator, or Authy.

עברית

להגדרת אימות רב-גורמי, עברו אל הגדרות חשבון > אבטחה ולחצו על הפעל MFA. תתבקשו לסרוק קוד QR באמצעות אפליקציית אימות לפי בחירתכם, כגון Google Authenticator, Microsoft Authenticator או Authy.

4.2 Backup Codes
English

Upon enabling MFA, ten single-use backup codes will be generated. Store these in a secure location. Each code can be used only once.

עברית

עם הפעלת ה-MFA, יופקו עשרה קודי גיבוי חד-פעמיים. שמרו אותם במקום מאובטח. כל קוד ניתן לשימוש פעם אחת בלבד.

4.3 Role-Based Access Permissions
English

SecureConnect Pro supports granular role-based access control (RBAC). Roles include: Owner (full access), Administrator (settings and user management), Editor (content creation and editing), and Viewer (read-only access).

עברית

SecureConnect Pro תומך בבקרת גישה מפורטת מבוססת תפקידים (RBAC). התפקידים כוללים: בעלים (גישה מלאה), מנהל (הגדרות וניהול משתמשים), עורך (יצירה ועריכת תוכן), ומציג (גישה לקריאה בלבד).
